# e-vue

> An electron-vue template

#### Build Setup

``` bash
# install dependencies
yarn

# serve with hot reload at localhost:9080
yarn run dev

# build electron application for production
yarn run build

#build dists
yarn dist

# lint all JS/Vue component files in `src/`
yarn lint --fix

```
